# Project Name

> Fan4stic

## Description 

A restaurant reviews webapp. 


## Table of Contents

1. [Usage](#Usage)
1. [Images](#Images)

## Usage

1. Install dependencies with 'npm install'
1. Seed database with 'npm run seed'
1. Compile with 'npm run webpack'
1. Launch with 'npm start'
